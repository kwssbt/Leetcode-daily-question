# [3296. 移山所需的最少秒数](https://leetcode.cn/problems/minimum-number-of-seconds-to-make-mountain-height-zero/)

**题目难度：Medium**

## 二分答案

对工人 $i$ ，完成工作量 $h_i$ 耗时

 $$t_i = \frac{workerTimes[i]\times h_i(h_i+1)}{2}$$

求 $ ans= max(t_i) $

每个工人的工作量可能不同，但 $t_i \le ans $ 一定成立，则有

$$ \frac{workerTimes[i]\times h(h+1)}{2} \le ans$$

解 $h_i$

$$ h_i \le \frac{-1+\sqrt{1+8\times \frac{ans}{  workerTimes[i]}}}{2}$$

对于一个 $ans$ 检查是否满足 

$$\sum_{i=0}^{n-1} h_i \ge  mountainHeight $$

由此可以二分答案求解

下界：`workerTimes.size()=1`,`workerTimes[0]=1`,`mountainHeight=1`
$$ans_m=1$$

上界：`workerTimes.size()=1`,`workerTimes[0]=1e6`,`mountainHeight=1e5`,$$ans_M=\frac{1e6\times1e5\times(1e5+1)}{2}<1e16$$

## 代码实现 (C++)
```cpp
class Solution {
public:
    long long minNumberOfSeconds(int mountainHeight, vector<int>& workerTimes) {
        function<bool(long long)>check=[&](long long x){
            long long cnt=0;
            for(int k:workerTimes){
                long long h=(sqrt(1.0+8.0*x/k)-1.0)/2.0;
                cnt+=h;
                if(cnt>=mountainHeight){
                    return 1;
                }
            }
            return 0;
        };
        long long l=1,r=1e16,mid,ans=r;
        while(r>=l){
            mid=(r+l)>>1;
            if(check(mid))ans=mid,r=mid-1;
            else l=mid+1;
        }
        return ans;
    }
};
```

- 时间复杂度: $O(N \log M)$
- 空间复杂度: $O(1)$
